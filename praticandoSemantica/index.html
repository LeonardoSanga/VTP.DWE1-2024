Elementos Semânticos

<article>

Representa um conteúdo independente, como uma postagem de blog ou artigo, que pode ser entendido por si só

Exemplo: https://www.w3schools.com/tags/tag_article.asp

<footer>

Define a área de rodapé para um documento ou seção.

Exemplo: https://www.w3schools.com/tags/tag_footer.asp

<nav>

Define um conjunto de links de navegação. O elemento destina-se apenas a blocos principais de links de navegação.

Exemplo: https://www.w3schools.com/tags/tag_nav.asp

<aside>

Define algum conteúdo além daquele em que é colocado. Deve estar indiretamente relacionado ao conteúdo circundante.

Exemplo: https://www.w3schools.com/tags/tag_aside.asp

<nav>

Define a área de cabeçalho de uma página ou seção, geralmente contendo títulos, logotipos e menu de navegação.

Exemplo: https://www.w3schools.com/tags/tag_header.asp

<section>

Define uma seção em um documento.

Exemplo: https://www.w3schools.com/tags/tag_section.asp

<details>

Especifica detalhes adicionais que o usuário pode abrir e fechar sob demanda. É frequentemente usado para criar um widget interativo que o usuário pode abrir e fechar.

Exemplo: https://www.w3schools.com/tags/tag_details.asp

<main>

Especifica o conteúdo principal de um documento.

Exemplo: https://www.w3schools.com/tags/tag_main.asp

<summary>

Define um cabeçalho visível para o elemento <details>. O título pode ser clicado para visualizar ou ocultar os detalhes.

Exemplo: https://www.w3schools.com/tags/tag_summary.asp

<figcaption>

Define uma legenda para um elemento <figure>. Pode ser colocado como o primeiro ou último filho do elemento <figure>.

Exemplo: https://www.w3schools.com/tags/tag_figcaption.asp

<mark>

Define o texto que deve ser marcado ou destacado.

Exemplo: https://www.w3schools.com/tags/tag_mark.asp

<time>

Define uma hora específica (ou data e hora).

Exemplo: https://www.w3schools.com/tags/tag_time.asp

<figure>

Especifica conteúdo independente, como ilustrações, diagramas, fotos, listagens de códigos, etc.

Exemplo: https://www.w3schools.com/tags/tag_figure.asp
